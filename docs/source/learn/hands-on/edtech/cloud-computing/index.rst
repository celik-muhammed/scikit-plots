Cloud Computing Labs
====================

These labs provide the learners with practical experience as they explore the
principals the cloud computing components of SaaS, PaaS, and IaaS.
The participants will learn how to configure a VPS (virtual private server),
install locally-hosted applications, use Nginx as a reverse proxy,
run Docker-based applications, learn how to create your own Docker images,
and configure a VPN server.

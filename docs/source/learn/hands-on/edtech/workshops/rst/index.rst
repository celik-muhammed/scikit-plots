****************************
RST Workshop - Germany 2020
****************************

This workshop has two components:

    #. Writing documentation using reStructuredText (RST)
    #. Setting up a Sphinx server

Afterwards, follow the instructions to set up a Sphinx server for
your documentation. Or, just build a Docker images using the template
``Dockerfile``. There are two variations:

    #. Debian - Well supported packages, stable
    #. Alpine - Small footprint

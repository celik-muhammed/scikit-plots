The lab walks you through the processes of setting up the tools to
write and compile C code. You will then create a C program to learn
about input and output in C.

The C programming environment for these labs consist of two components,
which are:

#. A code editor: |Visual Studio Code|
#. A C/C++ compiler: |MinGW|

Goals for Lab 1
================
During this lab, you will learn how to:

 #. **Setup** your C programming environment.
 #. **Compile** an ``.c`` source file to an ``.exe`` file.
 #. **Write** data to the console window
 #. **Read** input from the console window

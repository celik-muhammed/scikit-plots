C Programming
=============

.. include:: c-urls.rst

These labs provide the learners with practical experience as they learn
about C programming.

.. note:: Some labs are platform independent using Windows or POSIX (Linux), but
    other labs require POSIX.

    - See :ref:`Windows 10 C Environment` on how to set up Windows 10 to compile
      and execute C.
    - Write and compile code online at |ide.judge0.com| in a POSIX
      environment.
